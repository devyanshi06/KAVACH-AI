# KAVACH-AI: Secure Offline Copilot

## Overview
This is a secure, air-gapped AI assistant designed for ISRO network operations.

## How to Launch (The "One-Click" Method)
Instead of using manual terminal commands, simply double-click the **`start_vault.bat`** file in the root directory. This will:
1. Automatically activate the virtual environment.
2. Launch the Streamlit interface.
3. Open the dashboard in your default web browser.

## Team Protocols
- **ML Engineer:** Update `requirements.txt` whenever you add a new library.
- **Network Lead:** Place new knowledge base files in the `/vault` directory.
- **DevOps/Security:** Ensure all new code passes the offline air-gap test before deployment.
